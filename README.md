# My Steam Mod Script Generator
> Script generator for downloading mods using steamcmd

## How to run
```
go get fyne.io/fyne/v2
go run smsg.go
```

## How to use
Run programm
Enter link to mod or mode collection
After close? run:
```steamcmd +runscript script.txt```

## How to build
use fyne-cross
(fyne-cross <platform> [options])
